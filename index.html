Limiter

Spy_Tasker allows you to set process-specific CPU, RAM, and time limits. If any process exceeds its defined threshold, the system will terminate it and optionally log an alert. It's useful for handling untrusted apps, background drainers, and unwanted system hogs.

Security

The Security module (Authenticator) watches failed login attempts using Event Logs. If the attempts exceed a limit, it sends you email. It acts like a smart watchdog to protect your system when you're away.
